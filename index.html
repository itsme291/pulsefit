Sync workouts to Google Drive so Gemini can read them. Requires a Google Cloud Client ID.

Disconnected
  1. Go to the Google Cloud Console.
  2. Create a new project (e.g., "PulseFit").
  3. Search for Google Drive API and click Enable.
  4. Go to APIs & Services > OAuth consent screen:
    • User Type: External.
    • App name: PulseFit.
    • Developer email: your email.
    • Add scope: ../auth/drive.file (non-sensitive).
    • Add your own email to Test Users (CRITICAL).
  5. Go to Credentials > Create Credentials > OAuth client ID:
    • Application type: Web application.
    • Authorized JavaScript origins: http://localhost:8000 and/or your hosted site domain.
    • Authorized redirect URIs: http://localhost:8000 and/or your hosted site domain.
  6. Copy the Client ID (e.g. 12345-abc.apps.googleusercontent.com) and paste it above!
